## サービス概要
区エストは東京都内のイベントをゲーム感覚で楽しめるサービスです。
東京都内の様々なイベントに参加するきっかけを提供します。

## 使用技術
### バックエンド

- Ruby 3.1.0
- Rails 7.0.3

### インフラ

- Heroku
- PostgreSQL
- Amazon S3
